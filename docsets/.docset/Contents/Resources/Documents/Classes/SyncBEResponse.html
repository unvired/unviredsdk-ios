SyncBEResponse

@interface SyncBEResponse : NSObject <ISyncResponse>

Swift

class SyncBEResponse : NSObject, ISyncResponse

SyncBEResponse is the response for synchronous calls by the application to exchange data with the server.

SyncBEResponse provides the response code to indicate the status of the call. The response code is the http response code. Based on the code different output is provided.

Http response code is not 200: Failure. Connection with the server is not successful. The error message is returned by the method getErrorMessage() as a String.

Http response code is 200: Successful call. If auto save option is chosen the the incoming data is auto saved and the saved data is returned to the caller. Without auto save option the data is returned without saving. The format of the return data is same in both auto save required and not required cases.

For successful connections the return either can be a vector of InfoMessage objects or a hashtable of data business entities or both based on the response from the server.

Hashtable of business entities contain the business entity name as the key and a hashtable of structures as values. The hashtable of structures contains the header data structure as the key and the item data structures as values.
