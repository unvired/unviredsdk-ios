Logger

Objective-C

@interface Logger : NSObject

Swift

class Logger : NSObject

This class provides APIs for logging in the application. The following code block logs a message under the level ERROR.

  [Logger loggerWithlog:LEVEL_ERROR
              ClassName:[self class]
             MethodName:NSStringFromSelector(_cmd)
                    msg:@"This is an example for logging under the LEVEL_ERROR"];

  • +setLogLevel:

    Set a default log level for this Logger Class. Logs written with the log level lesser than the default log level of the logger will always get logged. For example: If you set the log level as LEVEL_DEBUG (9) then logs with LEVEL_IMPORTANT (7) and LEVEL_ERROR(8) get logged.
